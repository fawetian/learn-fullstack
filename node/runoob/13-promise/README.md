# Node.js Promise

> 来源: https://www.runoob.com/nodejs/nodejs-promise.html

## 核心概念

1. Promise 是异步编程的解决方案
2. 三种状态：pending、fulfilled、rejected
3. .then() 处理成功，.catch() 处理错误
4. Promise 链避免回调嵌套
