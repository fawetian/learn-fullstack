# Node.js 路由

> 来源: https://www.runoob.com/nodejs/nodejs-router.html

## 核心概念

1. 路由根据 URL 路径分发请求
2. 解析 URL 获取路径和查询参数
3. HTTP 方法（GET/POST）区分操作
4. 可构建简单的路由分发系统
