# Node.js Web 模块

> 来源: https://www.runoob.com/nodejs/nodejs-web-module.html

## 核心概念

1. http 模块创建 Web 服务器和客户端
2. url 模块解析和格式化 URL
3. querystring 模块处理查询字符串
4. net 模块创建 TCP 服务器

## 文件说明

| 文件名 | 类型 | 说明 |
|--------|------|------|
| example_1.js | JavaScript 代码 | 创建 HTTP 服务器示例 |
| example_2.html | HTML 页面 | Node.js 代码示例 |
| example_3.js | JavaScript 代码 | HTTP 客户端请求示例 |
| example_4.js | JavaScript 代码 | 模块导入示例 |

## 运行方法

确保已安装 Node.js，然后在终端执行：

```bash
cd 31-web-module
node example_1.js
```

```bash
cd 31-web-module
node example_3.js
```

```bash
cd 31-web-module
node example_4.js
```
