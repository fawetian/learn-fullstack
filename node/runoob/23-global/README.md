# Node.js 全局对象

> 来源: https://www.runoob.com/nodejs/nodejs-global-object.html

## 核心概念

1. global 是全局命名空间对象
2. process 对象提供进程信息和控制
3. Buffer 类无需 require 即可使用
4. console、setTimeout 等也是全局可用的
