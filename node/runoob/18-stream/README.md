# Node.js Stream

> 来源: https://www.runoob.com/nodejs/nodejs-stream.html

## 核心概念

1. Stream 是连续的数据流处理抽象
2. 四种类型：可读、可写、双工、转换
3. pipe() 方法连接多个流
4. 背压（backpressure）机制防止内存溢出
