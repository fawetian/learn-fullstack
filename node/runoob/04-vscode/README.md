# Node.js VS Code

> 来源: https://www.runoob.com/nodejs/nodejs-vscode.html

## 核心概念

1. VS Code 是 Node.js 开发的优秀编辑器
2. 安装 Node.js 扩展增强开发体验
3. 支持调试、智能提示和代码补全
4. 可配置 launch.json 进行调试

## 文件说明

| 文件名 | 类型 | 说明 |
|--------|------|------|
| example_1.js | JavaScript 代码 | 创建 HTTP 服务器示例 |

## 运行方法

确保已安装 Node.js，然后在终端执行：

```bash
cd 04-vscode
node example_1.js
```
