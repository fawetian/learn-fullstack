# Node.js GET/POST 请求

> 来源: https://www.runoob.com/nodejs/node-js-get-post.html

## 核心概念

1. GET 请求通过 URL 查询参数传递数据
2. POST 请求通过请求体传递数据
3. querystring 模块解析查询字符串
4. URL 模块处理 URL 路径和参数
