# Node.js Buffer

> 来源: https://www.runoob.com/nodejs/nodejs-buffer.html

## 核心概念

1. Buffer 用于处理二进制数据
2. Buffer 是固定大小的原始内存分配
3. Node.js 处理流数据时使用 Buffer
4. Buffer 与字符串可相互转换
