# Node.js 文件系统

> 来源: https://www.runoob.com/nodejs/nodejs-fs.html

## 核心概念

1. fs 模块提供文件系统操作
2. 同步方法（Sync）阻塞执行，异步方法非阻塞
3. 流式读写适合大文件处理
4. fs.promises 提供基于 Promise 的 API
