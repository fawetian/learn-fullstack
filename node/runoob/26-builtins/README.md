# Node.js 内置模块

> 来源: https://www.runoob.com/nodejs/nodejs-buildin-modules.html

## 核心概念

1. Node.js 提供丰富的内置模块
2. path、os、url 等常用模块
3. 无需安装直接 require 使用
4. 查阅官方文档了解全部模块
