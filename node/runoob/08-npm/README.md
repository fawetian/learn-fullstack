# NPM 使用介绍

> 来源: https://www.runoob.com/nodejs/nodejs-npm.html

## 核心概念

1. NPM 是 Node.js 的包管理工具
2. package.json 管理项目依赖和脚本
3. npm install 安装依赖包
4. node_modules 存放已安装包
