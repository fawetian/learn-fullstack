# Node.js RESTful API

> 来源: https://www.runoob.com/nodejs/nodejs-restful-api.html

## 核心概念

1. RESTful API 基于 HTTP 方法定义操作
2. GET 获取、POST 创建、PUT 更新、DELETE 删除
3. JSON 作为数据交换格式
4. Express 快速构建 REST API

## 运行方法

确保已安装 Node.js，然后在终端执行：

```bash
cd 33-restful
node example_2.js
```

```bash
cd 33-restful
```

