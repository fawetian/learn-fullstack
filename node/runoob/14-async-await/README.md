# Node.js async/await

> 来源: https://www.runoob.com/nodejs/nodejs-async-await.html

## 核心概念

1. async/await 是 Promise 的语法糖
2. async 函数返回 Promise 对象
3. await 暂停执行等待 Promise 解决
4. 使异步代码看起来像同步代码
