## 核心概念

1. 结合 HTTP 模块和文件系统构建应用
2. 处理 JSON 数据交换
3. 模块化组织代码结构
4. 实战演练 Node.js 基础能力

## 文件说明

| 文件名 | 类型 | 说明 |
|--------|------|------|
| example_1.js | JavaScript 代码 | 创建 HTTP 服务器示例 |
| example_2.js | JavaScript 代码 | 创建 HTTP 服务器示例 |

## 运行方法

确保已安装 Node.js，然后在终端执行：

```bash
cd 28-simple-app
node example_1.js
```

```bash
cd 28-simple-app
node example_2.js
```
