# Node.js 概念

> 来源: https://www.runoob.com/nodejs/nodejs-intro.html

## 核心概念

1. Node.js 是服务端 JavaScript 运行环境
2. Node.js 基于事件驱动和非阻塞 I/O
3. V8 引擎执行 JavaScript 速度非常快
4. Node.js 适合构建高性能网络应用
