# Node.js Express 框架

> 来源: https://www.runoob.com/nodejs/nodejs-express-framework.html

## 核心概念

1. Express 是轻量级的 Web 应用框架
2. 路由系统处理不同 URL 和方法
3. 中间件机制处理请求和响应
4. 模板引擎渲染动态页面
