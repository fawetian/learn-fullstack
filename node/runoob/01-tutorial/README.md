# Node.js 教程

> 来源: https://www.runoob.com/nodejs/nodejs-tutorial.html

## 核心概念

1. Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行时环境
2. Node.js 使用事件驱动、非阻塞 I/O 模型
3. Node.js 适合高并发、I/O 密集型应用场景
4. Node.js 采用单线程模型，通过事件循环处理并发
