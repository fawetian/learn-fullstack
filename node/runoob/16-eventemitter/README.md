# Node.js EventEmitter

> 来源: https://www.runoob.com/nodejs/nodejs-event.html

## 核心概念

1. EventEmitter 是发布-订阅模式的实现
2. on() 注册监听器，emit() 触发事件
3. 事件驱动是 Node.js 的核心设计
4. 可构建自定义事件系统
