# Node.js 模块导入

> 来源: https://www.runoob.com/nodejs/nodejs-commonjs-esm.html

## 核心概念

1. CommonJS（require/module.exports）与 ES Modules（import/export）
2. ESM 支持静态分析和 tree-shaking
3. .mjs 文件或 type: module 启用 ESM
4. 两种模块系统的互操作性
