# Node.js util 模块

> 来源: https://www.runoob.com/nodejs/nodejs-util-module.html

## 核心概念

1. util.inherits 实现构造函数继承
2. util.inspect 调试输出对象
3. util.promisify 回调转 Promise
4. util.format 格式化字符串
