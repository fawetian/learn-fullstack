# Node.js 常用工具

> 来源: https://www.runoob.com/nodejs/nodejs-util.html

## 核心概念

1. util 模块提供实用工具函数
2. util.promisify 将回调转换为 Promise
3. util.inherits 实现原型继承
4. util.inspect 格式化对象输出
