# Node.js 异步编程

> 来源: https://www.runoob.com/nodejs/nodejs-asynchronous.html

## 核心概念

1. JavaScript 是单线程语言，异步处理避免阻塞
2. 异步编程方式：回调、Promise、async/await
3. Node.js 大量 API 采用异步设计
4. 异步操作完成后通过事件通知主线程
