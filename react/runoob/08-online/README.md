# 08-online: React 在线练习

来源: https://www.runoob.com/react/react-online.html

## 核心概念

1. **在线练习平台** — 菜鸟教程提供无需环境配置的 React 代码运行环境，适合快速测试和验证代码片段。
2. **CodePen / StackBlitz** — 第三方在线编辑器，支持 React 项目模板，可保存和分享代码。
3. **即时验证** — 在线环境适合验证语法和概念理解，不需要等待本地构建过程。
4. **React Playground** — 官方和第三方提供的 React 沙盒环境，支持 JSX 编译和实时预览。
5. **学习路径建议** — 在线环境适合入门阶段，但真实项目开发需要本地工具链支持（包管理、Git、构建优化）。

## 文件说明

本章无代码文件，仅提供在线练习资源指引。

## 运行/学习方法

本章是工具介绍章节。建议按以下路径学习：

1. 在菜鸟教程在线环境中理解基础语法和概念
2. 在本地使用 Vite 创建项目（参见 `04-install-vite`）
3. 跟随 `33-blog-project` 到 `42-blog-deploy` 完成完整博客项目

在线练习资源：
- 菜鸟教程在线编辑器
- CodePen（React 模板）
- StackBlitz（Vite React 模板）
