# 02-intro: React 简介

来源: https://www.runoob.com/react/react-intro.html

## 核心概念

1. **React 起源** — 由 Facebook 工程师 Jordan Walke 创建，2013 年开源，目前由 Meta 和社区维护。
2. **为什么使用 React** — 组件化提高复用性、虚拟 DOM 提升性能、丰富的生态系统、跨平台（React Native）。
3. **React 核心特点** — 声明式 UI、组件化架构、一次学习到处编写（Web/Native/Desktop）。
4. **JSX 语法** — 在 JavaScript 中编写类似 HTML 的语法，编译后转换为 `React.createElement` 调用。
5. **生态系统** — React 本身只关注视图层，配合 React Router、Redux、Next.js 等库可构建完整应用。
