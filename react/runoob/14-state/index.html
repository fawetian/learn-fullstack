14-state - React 状态管理演示

example_1: 箭头函数语法（ES6）

const f = ([参数]) => 表达式

箭头函数是 ES6 特性，React 组件中广泛使用，自动绑定外层 this。

example_2: 箭头函数多种形式

(参数1, 参数2) => { 函数声明 }

单行表达式可省略 return 和 {}；无参数时必须写 ()。

example_3: 类组件中 setInterval 的 this 问题（错误示范）

普通 function 中 this 指向调用上下文（window 或 undefined），不是组件实例。

example_4: 使用 that 变量保存 this（传统方案）

let that = this;
this.timerID = setInterval(function () {
  return that.tick();
}, 1000);

example_5: 错误示范 - 普通函数内 this 指向 window

this.timerID = setInterval(function(){
  return this.tick();  // this 指向 window，报错！
}, 1000);
