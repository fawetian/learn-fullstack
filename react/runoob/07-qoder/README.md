# 07-qoder: Qoder 编辑器

来源: https://www.runoob.com/react/react-qoder.html

## 核心概念

1. **Qoder 编辑器** — 菜鸟教程提供的在线代码编辑器，支持 React 代码的在线编写和实时预览。
2. **在线开发环境** — 无需本地安装 Node.js 和编辑器，浏览器中即可完成 React 代码的编写和运行。
3. **即时反馈** — 修改代码后立即看到效果，适合初学者快速验证想法和理解概念。
4. **学习辅助** — 在线编辑器降低了环境配置的门槛，让学习者专注于 React 本身而非工具链。
5. **局限性** — 在线编辑器功能相对简单，不适合复杂项目开发，掌握后应迁移到本地 VS Code + Vite 环境。

## 文件说明

本章无代码文件，仅提供编辑器使用说明。

## 运行/学习方法

本章是工具介绍章节，了解菜鸟教程的在线编辑器使用方法。访问菜鸟教程对应页面，可以直接在浏览器中修改和运行 React 代码示例。建议在本地环境搭建完成后，使用 VS Code + Vite 进行实际项目开发。
