# 06-vscode: VS Code 编辑器配置

来源: https://www.runoob.com/react/react-vscode.html

## 核心概念

1. **VS Code** — 微软开发的免费开源代码编辑器，通过插件扩展支持 React 开发，是前端开发的主流选择。
2. **ES7+ React/Redux/React-Native snippets** — 提供 React 代码片段（如 `rfce` 快速生成函数组件），大幅提升编码效率。
3. **Prettier** — 代码格式化工具，自动统一代码风格（缩进、引号、分号等），团队协作必备。
4. **ESLint** — 静态代码分析工具，检测潜在错误和不规范写法，帮助写出更健壮的 React 代码。
5. **自动导入** — VS Code 的 JavaScript/TypeScript 语言服务支持自动补全和模块导入，减少手动输入。

## 运行/学习方法

本章是开发环境配置章节。建议安装以下 VS Code 插件以提升 React 开发效率：

- **ES7+ React/Redux/React-Native snippets** — 快速生成组件模板
- **Prettier - Code: formatter** — 自动格式化代码
- **ESLint** — 代码质量检查
- **Auto Rename Tag** — 自动同步修改 HTML/JSX 标签对

无需运行代码，配置完成后即可在后续章节中体验效率提升。
