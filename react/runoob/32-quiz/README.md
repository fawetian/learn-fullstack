# 32-quiz: React 测验

来源: https://www.runoob.com/react/react-quiz.html

## 核心概念

1. **知识检验** — 通过测验题检验对 React 核心概念（JSX、组件、Props、State、生命周期、Hooks）的掌握程度。
2. **常见陷阱** — 测验通常涵盖 `setState` 异步性、闭包陷阱、依赖数组遗漏、`key` 属性错误等常见面试考点。
3. **JSX 规则** — 测验可能涉及 JSX 语法限制（只能返回一个根元素、className 代替 class、style 是对象等）。
4. **Hooks 规则** — 只在顶层调用、只在 React 函数中调用、不能嵌套在 if/循环 中。
5. **性能优化** — 测验可能涉及 `React.memo`、`useMemo`、`useCallback` 的适用场景和使用条件。

## 运行/学习方法

本章是自我检测章节。完成测验的方法：

1. 访问菜鸟教程对应页面，完成所有测验题目
2. 记录错题，回到对应章节重新学习（如 Hooks 题目错误 → 复习 `30-hooks`）
3. 将易混淆概念整理成笔记（如 `useEffect` vs `useLayoutEffect`）

建议完成博客项目（`33-blog-project` 到 `42-blog-deploy`）后再做测验，实战中的理解更深刻。
