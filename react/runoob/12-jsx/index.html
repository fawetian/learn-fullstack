12-JSX - React JSX 语法演示

本页面将 JSX 代码片段整合为可运行的完整示例。打开浏览器控制台可查看日志。

example_4: 自定义组件命名规则

React 组件名必须以大写字母开头，否则会被当作普通 HTML 标签处理。

example_5: React Fragment（空标签）

Fragment 允许组件返回多个元素，而不向 DOM 添加额外节点。

example_6: JSX 语法规则

演示 className、自闭合标签、JSX 注释等基础规则。

example_7: JSX 中嵌入 JavaScript 表达式

在 JSX 中使用 {} 包裹任意有效的 JavaScript 表达式。

example_8: 条件渲染

使用三元运算符和逻辑与运算符进行条件渲染。

example_9: 列表渲染

使用 map() 方法遍历数组渲染列表，key 属性必须唯一且稳定。

example_12: CSS Modules 样式导入

演示 CSS Modules 用法（实际需构建工具支持，此处仅展示代码结构）。

import styles from './styles/Card.module.css';

example_14: 组件接收 props

解构 props 并使用 CSS Modules。

example_15: JSX 中的注释

JSX 注释使用 { /* 注释 */ } 语法。

example_16: JSX 必须有根元素

JSX 表达式只能有一个根元素，多个元素必须用包裹元素或 Fragment 包裹。

example_17: 使用包裹元素作为根

使用 section 等标签包裹多个子元素。

example_18: 错误的内联样式写法

JSX 中 style 属性必须接收一个 JavaScript 对象，而非 CSS 字符串。

example_19: 正确的内联样式写法

正确写法：style = {{ fontSize: 12 }}，外层 {} 是 JSX 表达式，内层 {} 是 JS 对象。
