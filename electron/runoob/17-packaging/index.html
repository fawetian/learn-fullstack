外部资源（extraResources 演示）

打包命令

基础打包

# 使用默认配置（electron-builder.yml）打包当前平台 npm run build

完整配置打包

# 使用完整配置（多平台/多格式/图标） npm run build:full

指定平台

npm run build:mac npm run build:win npm run build:linux

不打包 asar（源码可见）

# 打包后可在 dist/win-unpacked/resources/app/ 直接看到源码 npm run build:no-asar
